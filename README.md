# PromptForge (@ThePromptForgeAiBot)

An AI-powered Telegram bot designed to generate professional image prompts on demand.

## Deployment
This bot is configured to be deployed on **Render** using a GitHub repository.

## Setup
1. Clone this repository.
2. Create a `.env` file using `.env.example`.
3. Install requirements: `pip install -r requirements.txt`.
4. Run the bot: `python bot.py`.
